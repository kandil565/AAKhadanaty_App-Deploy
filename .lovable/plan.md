

# أخدماتي - Service Booking & Management System

## Overview
A complete Arabic RTL service booking platform for home maintenance and personal services, built with Egyptian Pound (EGP) currency, realistic Arabic content, and a professional graduation-project-ready design.

---

## 1. Theme & Styling Setup
- Apply the uploaded Arabic color scheme (blue primary, orange accent)
- Import Cairo & Tajawal Arabic fonts
- Set RTL direction globally
- Dark mode support

## 2. Pages & Navigation

### Public Pages:
- **الرئيسية (Home)** — Hero section, featured services, testimonials, stats, CTA
- **الخدمات (Services)** — Full grid of 18+ services with filtering by category
- **تفاصيل الخدمة (Service Details)** — Description, provider info, reviews, booking button
- **احجز الآن (Booking Page)** — Date/time picker, service selection, booking form with validation
- **تسجيل الدخول (Login)** — Login form with demo credentials
- **إنشاء حساب (Register)** — Registration form

### Dashboard Pages:
- **لوحة التحكم (Dashboard)** — Stats cards (total bookings, revenue in EGP, ratings), recent bookings chart
- **حجوزاتي (My Bookings)** — Booking list with status workflow (Pending → Confirmed → Completed → Canceled)
- **الملف الشخصي (Profile)** — User info management
